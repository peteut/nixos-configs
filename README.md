## NixOS Configuration

My configuration files for NixOS.
Uses [deploy-rs](https://github.com/serokell/deploy-rs) for deployments. for deployments.

### Deployment

Update  _my-node_ by invoking `LOCAL_KEY=nix.key deploy .#my-node`.
